Fractional Knapsack

You are given a knapsack with a maximum capacity of 200 units and a list of 4 items, each with an associated profit and weight:
• Item 1 — Profit: 12, Weight: 70
• Item 2 — Profit: 4, Weight: 30
• Item 3 — Profit: 7, Weight: 50
• Item 4 — Profit: 20, Weight: 90

Algorithm Steps
1. For each item, compute ratio = profit / weight
2. Sort all items in descending order of this ratio
3. Initialize totalProfit = 0, remainingCapacity = W
4. For each item in sorted order:
  a. If item's weight ≤ remainingCapacity:
    -Take the whole item.
    -Add its full profit to totalProfit.
    -Subtract its weight from remainingCapacity.
  b. Else:
    -Take fraction = remainingCapacity / item's weight
    -Add fraction x profit to totalProfit.
    -Knapsack is now full. Break.
5. Return totalProfit.
